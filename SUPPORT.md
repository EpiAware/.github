# Support

This applies across the [EpiAware](https://github.com/EpiAware)
organisation's packages.

## Where to ask

- **Usage questions and discussion**: open a discussion or issue on the
  relevant package repository.
- **Bugs and feature requests**: open an issue using the templates on the
  relevant repository.
- **General Julia help**: the
  [Julia Discourse](https://discourse.julialang.org) and the
  [Julia Slack](https://julialang.org/slack/) are good places for broader
  questions.

Before asking, please search existing issues and discussions in case your
question has already been answered.

## Contributing

If you would like to help, see [CONTRIBUTING.md](CONTRIBUTING.md).
